Cellular-Automata
=================

LWJGL Based Life-Like Cellular Automata

## Keys

Commands

    ESC   - Quit
    F     - Toggle full screen
    C     - Toggle color cycling
    T     - Toggle trails
    R     - clear and randomize cells
    SPACE - pause simulation
    -     - decrease cell size
    +     - increase cell size

Automata

    1     - Conway's Life
    2     - 34 Life
    3     - Diamoeba
    4     - High Life
    5     - Morley
    6     - 2 x 2
    7     - Seeds
    8     - Day and Night
    9     - Coral
    0     - Replicator
